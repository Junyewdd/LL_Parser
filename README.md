# Lexical_Analyzer-Parser

1. 정의되어 있지 않은 식별자 쓸 때
- 처음 식별자 : unknown 처리
  -> (Error) "정의되지 않은 변수({ident_name})가 참조됨"
- 정의되어 있지 않은 식별자를 한 실행문에 여러 개 쓸 때
  -> 모든 에러 메시지를 띄우기
- 값이 unknown인 식별자를 쓰면 -> unknown 
  -> (OK)

2. 중복 연산자 제거
- 같은 연산자 반복 : 하나로 인식
  -> (Warning) "중복 연산자(+) 제거"
- 다른 연산자 등장 : 처음 연산자만 인식
  -> (Error) "한 종류 이상의 연산자가 연속으로 쓰임"
